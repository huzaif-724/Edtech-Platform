## Getting Started

1. Clone the repository:
```bash
https://github.com/huzaif-724/BlogApp.git
```

2. Navigate to the project directory:
```bash
cd BlogApp
```

3. Install dependencies:
```bash
# For backend
cd backend
npm install

# For frontend 
cd ../frontend
npm install
```

4. Start the development servers:
```bash
# Start backend server
cd backend
npm start

# Start frontend server
cd ../frontend
npm start
```
